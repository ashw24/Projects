Lab 3: ASCII-risks (Asterisks)
-----------
DESCRIPTION
In this lab, the user enters a number when the prompt asks for it. The program then generates a pyramid of asteriks with tabs between them in respect to the number that is entered. 
-----------
FILES
Lab3.asm
This file contains the code required for this lab to run.  
-----------
INSTRUCTIONS
This software requires the MIPS Assembler and Runtime Simulator to function (MARS). If the test case is entered as input, MARS is used to execute it.
