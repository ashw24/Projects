Lab 2: Simple Data Path
-----------
DESCRIPTION
In this lab, the user enters a two numbers. The numbers are stored and the first number is rotated to the left by the second number.
-----------
FILES

ReadMe
This file contains the information that explains the assignment
-
Lab4.asm
This file includes the MML logic of the lab.
-----------
INSTRUCTIONS
This program is intended to be run using the MML software. Enter the value in the keypad and select a register to store it in the registers using the switches and by pressing the update button. The first number is rotated to the left by the second number.
